OpenChime

The open source school bell system.

OpenChime is an Open Source Raspberry Pi based school bell system. Most systems are expencive, unintuitive, and are limited in what they can do. While other systems get the job done, OpenChime can do all that and more.

  • Open Source

  • Raspberry Pi Based

  • Customizable Sounds

  • Accurate time

  • Secure Web Interface

  • IOT Integration
